# Card
卡片组件

### 示例
<e-card imgSrc="/1.jpeg" summary="0到1快速构建自己的后台管理系统"></e-card>

### 代码
```html
<e-card imgSrc="/1.jpeg" summary="0到1快速构建自己的后台管理系统"></e-card>
```

### Attributes
| 参数 | 说明 | 类型 | 是否必要 | 默认值 |
| --- | --- | --- | --- | --- |
| width | 卡片的宽度 | Number | false | - |
| imgSrc | 图片资源的只 | String | true | - |
| imgHeight | 图片的高度 | Number | false | - |
| summary | 卡片概要 | String/Slot | false | - |
| footer | 卡片底部 | Slot | false | - |
